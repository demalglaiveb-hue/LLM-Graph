# Contributing to LLM-Graph

Thank you for your interest in contributing!

## How to contribute

1. **Open an Issue** to discuss your idea or problem.
2. **Fork** the repository.
3. Create a **feature branch** from `main`.
4. Implement your changes and add/update tests or examples if needed.
5. Submit a **Pull Request** with a clear description.

For significant changes, please open an RFC first using
`.github/ISSUE_TEMPLATE/rfc.md`.

## Coding & Spec Guidelines

- Schemas should follow JSON Schema draft 2020-12.
- Changes to spec must update `SPEC_v0.2.0.md` and `CHANGELOG.md`.
- Strive for backward compatibility in minor releases.
