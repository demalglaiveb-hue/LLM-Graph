# LLM-Graph Roadmap (Draft)

This roadmap is indicative and will evolve with community feedback.

## Phase 1 — Consolidation (0–3 months)

- Stabilize Web / Document / Product / App Profiles for v0.2.x
- Improve online validator (multi-profile support)
- Add more real-world examples (web, document, product, app)

## Phase 2 — Ecosystem (3–6 months)

- WordPress plugin (stable)
- Shopify integration (stable)
- Webflow / Framer snippets
- Local CLI validator & linter
- Converters from OpenGraph / schema.org

## Phase 3 — Enterprise (6–12 months)

- Document Profile best practices for internal knowledge bases
- Recommended patterns for large catalogs (Product Profile)
- Governance and RFC process refinement
- Early adopters & certification program

## Phase 4 — Standardization (12+ months)

- v1.0.0 of the core spec
- External working group
- Alignment with existing standards bodies
- Reference implementations in multiple languages
